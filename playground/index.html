      Waterfall CAD Playground

      An in-browser playground for Waterfall CAD, a declarative Haskell CAD library built on OpenCASCADE. Code is compiled and run entirely in your browser in WASM.

      • Waterfall CAD API docs on Hackage
      • Source on GitHub
      • Made possible by the GHC WASM backend, by Cheng Shao et al.
